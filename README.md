# bearer-auth

Created a server authenticatiing users with Bearer authentication

## Installation

- `git clone https://github.com/mhendricks96/bearer-auth.git`

- `cd bearer-auth`

- `nmp i`

## Usage

- start server: `npm start`

- test server: `npm test`

## Routes

- `signup`: POST

- `signin`: POST

- `users`: GET

- `secret`: GET
